# CSCI 3000: Computer Communications Networks (CCN)
**Term:** Fall 2025  
**Level:** Upper Division Undergraduate  

---

## Course Philosophy
This course explores the architectural, algorithmic, and systemic foundations of modern computer networks, balancing theory with implementation.

---

## Curriculum Overview

### Phase 1: Architectural Foundations
- Internet Architecture
- Application Layer
- TLS Security

### Phase 2: Transport Layer
- Reliable Data Transfer
- TCP Congestion Control

### Phase 3: Network Layer
- Routing Algorithms
- IPv6

### Phase 4: Link Layer
- Switching & VLAN
- Wireless

### Phase 5: Security & SDN
- Firewalls
- QoS
- Automation
